## Description

A SDL2-based reimagining of the classic Sokoban puzzle game, rendered in an 8-bit style.

# Usage and Dependencies

**Dependency:** SDL2 must be installed on your system before building.

To compile and launch the game, run:

```bash
$> make
```

This will build the project and immediately start the game, dropping you into the first puzzle.

## Controls

| Key | Action         |
|-----|----------------|
| `W` | Move Up        |
| `S` | Move Down      |
| `D` | Move Right     |
| `A` | Move Left      |
| `R` | Restart Level  |
| `Q` | Quit Game      |
